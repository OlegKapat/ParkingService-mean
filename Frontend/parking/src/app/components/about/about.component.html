Про сервіс

  • tocПравила для орендарів
    1. Для користування сервісом потрібна реєстрація.
    2. З підключених до сервіса паркінгів вибираєте для вас зручний.
    3. Бронюєте з наданих варіантів на неохідний термін.
    4. Очікуєте підтверження власника паркінгу.
    5. Після підтверження вам надсилається лист на вказану електронну адресу.
    6. Телефонуєте власнику паркомісця і домовляєтесь про деталі.
    7. Всі розрахунки тільки з власником.
  • tocПравила для арендодавців
    1. Якщо ваш паркінг не зареєстрований в сиcтемі напишіть нам.
    2. Зареєструйтесь в системі.
    3. Прикріпіть файл документа який підтвержує право власності на паркомісце.
    4. Отримавши лист на електронну почту про бронювання вашого місця, дайте підтверження чи відхиліть пропозицію.
    5. Після підтверження ваш номер телефону буде надісланий заявникові для подальших домовленностей.

Сервіс з бронювання паркомісць.

Ми не надаємо послуги. Ми тільки допомагаємо знайти зацікавлених сторін для взаємовигідної співпарці. Якщо у вас є пропозиція чи питання пишіть нам і наша підтримка звяжеться з вашим представником
